원더걸스, 오는 7월 5일 싱글 'Why So Lonely'로 전격 컴백
데뷔 10년 만의 첫 자작 타이틀곡 'Why So Lonely' 확고한 뮤지션 입지 입증, 박진영 아닌 첫 자작 타이틀 곡

- 원더걸스, 첫 자작 타이틀곡 담은 싱글 'Why So Lonely'로 7월 5일 전격 컴백.
- 원더걸스의 음악적 역량 엿보는 타이틀곡 'Why So Lonely', 원더걸스 표 음악 담았다.
- 수식어 필요 없는 원더걸스, 자작 타이틀곡 'Why So Lonely'로 컴백! JYP 수장 박진영 곡 아닌 자작 타이틀 곡.

원더걸스가 새 싱글 'Why So Lonely'로 오는 7월 5일 전격 컴백한다.

원더걸스의 새 싱글 'Why So Lonely'에는 동명의 타이틀곡 'Why So Lonely'와 '아름다운 그대에게' 'Why So Lonely' 총 3곡이 수록됐다.

지난해 정규앨범 'REBOOT'를 통해 멤버 작사 작곡 능력을 자랑하며 감각적인 스타일로 4인조 밴드 변신에 성공한 원더걸스가 한층 더 업그레이드된 연주 실력과 음악으로 대중들을 찾는다.

타이틀곡 'Why So Lonely'는 원더걸스의 첫 자작 타이틀곡으로, 데뷔 이래 9년 동안 박진영이 만든 노래로 타이틀 곡으로 삼아온 원더걸스가 박진영의 품을 떠나 진정한 아티스트로써 성장해 내놓은 값진 결과물이다.

'Why So Lonely'는 원더걸스가 처음 시도하는 레게팝 장르의 곡으로, 중독성 있는 기타리프와 다채로운 리듬의 변화에 따른 분위기의 전환이 매력적인 노래다.

'너도 나만큼 빠져야 하는데 ' '너만 멋져 내 마음만 다쳐' 등 연인사이에 느끼는 시니컬한 내용을 사랑스럽고 위트 있는 가사로 표현해냈다.

수록곡 '아름다운 그대에게'는 작곡가 Frants와 원더걸스 멤버들이 의기투합하여 만든 곡으로 70년대 밴드 사운드의 영향을 받아 작업됐다.

사람 사이의 관계가 점점 차가워지는 디지털 시대에서 서로를 좀 더 아껴주고 사랑하는 세상이 되길 바라는 마음을 담았다.

또 다른 수록곡 'Sweet&Easy'는 여름에 듣기 좋은 팝락 장르의 곡으로, 제목처럼 달콤하고 쉬운 멜로디와 가사 그리고 리드미컬한 기타가 인상적이다.

걸그룹으로 시작해 밴드 변신에 성공한 원더걸스는 이번 디지털 싱글을 통해 더욱 성장하고 발전된 음악적 스펙트럼을 보여주며 확실한 존재감을 드러낼 전망이다.

데뷔 10년 차에도 끊임없이 진화하는 '최정상급 뮤지션 원더걸스'의 무게감 있는 행보에 기대가 모아진다.
